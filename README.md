AI-Powered Sign Language Translation for Inclusive Communication
A real-time system that translates sign language gestures into text and audio output using computer vision and deep learning.
Tech Stack: Python, TensorFlow, MediaPipe, WebRTC, MERN Stack
Features:
1.Real-time gesture recognition at 30fps
2.Sub-500ms response latency
3.Text and audio output
4.Modular backend API architecture
